# Handbook of Small Data Sets
This is a github repository of the datasets used in Hand, D., Daly, F., McConway, K., Lunn, D., Ostrowski, E. (1993). A Handbook of Small Data Sets. New York: Chapman and Hall/CRC.

## CRC-Press Dataset Release

CRC-Press released the datasets on their website.
These datasets can be downloaded publically.
The dataset filenames are as it appears in the Data Structure Index, pages 413 - 436 of the text.
(Provided as a PDF document in this repository)

Source: https://www.crcpress.com/A-Handbook-of-Small-Data-Sets/Hand-Daly-McConway-Lunn-Ostrowski/p/book/9780412399206
Last updated according to CNC-Press on 30 January 2008.
Downloaded on 23/03/2018.
